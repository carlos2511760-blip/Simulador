# 🌌 Antigravity Physics Simulator

Um simulador de física completo e interativo para web, focado em educação e visualização de fenômenos físicos em tempo real.

## 🚀 Funcionalidades

O simulador está dividido em **8 módulos fundamentais**:

1.  **Mecânica Clássica**: Leis de Newton, lançamentos de projéteis, colisões elásticas e inelásticas, planos inclinados e pêndulos.
2.  **Gravidade e Órbitas**: Simulação de N-corpos, sistemas solares, estrelas binárias e o problema caótico dos três corpos.
3.  **Energia**: Transformação entre energia cinética e potencial (gravitacional e elástica) com gráficos em tempo real.
4.  **Termodinâmica**: Comportamento de gases ideais, distribuição de velocidades de Maxwell-Boltzmann e condução térmica.
5.  **Ondas e Vibrações**: Ondas transversais e longitudinais, padrões de interferência e ondas estacionárias (nós e antinós).
6.  **Eletromagnetismo**: Cargas elétricas, vetores de campo, linhas de força, capacitores e potenciais elétricos.
7.  **Fluidos**: Empuxo (Arquimedes), vasos comunicantes, perfil de velocidade em fluxo laminar e pressão hidrostática.
8.  **Física de Materiais**: Corpos macios (Soft Bodies), estruturas de treliça (truss), tensões estruturais e fraturas.

## 🛠 Tecnologias

-   **HTML5 Canvas**: Para renderização de alta performance.
-   **Vanilla JavaScript**: Motor de física customizado com integração de Verlet e Euler.
-   **Vanilla CSS**: Design moderno com glassmorphism e sistema de temas escuros.

## 📦 Como usar o GitHub Desktop para Deploy

Para colocar seu simulador online no GitHub Pages:

1.  **Abra o GitHub Desktop**: Clique em "File" -> "Add Local Repository" e selecione a pasta `C:\Users\coord\OneDrive\Área de Trabalho\Simulador_Junior`.
2.  **Commit**: Escreva "Initial commit: Antigravity Physics Simulator" no campo de sumário e clique em "Commit to main".
3.  **Publish**: Clique em "Publish repository" para criar o repositório no seu GitHub.
4.  **Deploy (GitHub Pages)**:
    - Vá ao seu repositório no site do GitHub.
    - Clique em **Settings** -> **Pages**.
    - Em "Build and deployment", selecione a branch `main` e a pasta `/(root)`.
    - Clique em **Save**. Em alguns minutos, seu link será gerado (ex: `https://seu-usuario.github.io/Simulador_Junior/`).
